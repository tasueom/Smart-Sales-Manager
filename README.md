# Smart Sales Manager

판매 데이터를 손쉽게 수집·관리하고 다양한 차트로 매출 인사이트를 제공하는 Flask 기반 웹 애플리케이션입니다.

## 주요 기능

### 1. 매출 등록 및 관리
- 개별 거래를 직접 입력해 판매일자, 품목, 수량, 단가, 총액을 저장할 수 있습니다.
- 등록된 매출은 목록에서 페이지네이션과 함께 확인하고, 수정·삭제로 즉시 조정할 수 있습니다.

### 2. CSV 일괄 업로드
- 지정된 포맷(`sale_date,item_name,quantity,unit_price`)의 CSV 파일을 업로드하면 여러 건의 매출을 한 번에 적재합니다.
- 업로드 시 누락 데이터는 제거하고 형식을 검증해 안정적으로 저장합니다.

### 3. 매출 분석 대시보드
- 조회 월을 선택하면 해당 월의 데이터를 기준으로 일자별·품목별 차트와 비중 파이를 제공합니다.
- 품목별 총매출/판매수량의 복합 차트(bar+line)와 상위 3개 품목 요약 테이블로 핵심 정보를 빠르게 파악할 수 있습니다.
- 데이터가 없을 경우 안내 메시지를 표시해 상태를 즉시 확인할 수 있습니다.

### 4. 데이터 내보내기
- 현재 저장된 매출 전체를 Excel(.xlsx) 파일로 다운로드할 수 있어 회계 처리나 외부 보고에 활용하기 편리합니다.

## 기술 스택

- **Backend**: Flask (Python), pandas
- **Database**: MySQL, mysql-connector-python
- **Frontend**: HTML, CSS, JavaScript
- **시각화 라이브러리**: Chart.js

## 설치 및 실행

### 필수 패키지 설치
```bash
pip install flask pandas mysql-connector-python openpyxl
```

### 실행
```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 사용 방법

1. **홈**: 서비스 메뉴를 확인하고 주요 기능으로 이동합니다.
2. **매출 목록**: 등록된 매출을 확인하고, 개별 건에 대해 수정·삭제를 수행합니다.
3. **매출 입력 & CSV 업로드**: 단건 입력 또는 CSV 파일 업로드로 데이터를 추가합니다.
4. **매출 분석**: 조회 월을 선택해 일자별 매출, 품목별 매출/비중 차트, Top 3 품목 정보를 확인합니다.
5. **Excel 다운로드**: 언제든지 전체 매출 데이터를 엑셀 파일로 내보냅니다.
